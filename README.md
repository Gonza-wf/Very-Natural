# Very Natural - Auténtico Cuidado de la Piel

Bienvenido al repositorio de **Very Natural**, una landing page premium y bilingüe (ES/EN) para productos de cuidado de la piel artesanales y orgánicos.

## Características Principales

*   **Bilingüe (Español / Inglés):** Soporte multi-idioma integrado mediante JavaScript sin necesidad de recargar la página.
*   **Catálogo Dinámico:** El catálogo se genera a partir de datos estructurados en JavaScript, permitiendo el fácil agregado de nuevos productos.
*   **Filtros de Categoría:** Sistema de vistas elegante para categorías (Jabones, Mascarillas, Kits, Novedades) y productos.
*   **Diseño Premium y Responsivo:** Diseñado desde cero para lucir espectacular en dispositivos móviles y de escritorio.
*   **Integración con WhatsApp:** Botones de compra directos que abren un mensaje pre-armado en WhatsApp para facilitar las conversiones.
*   **Animaciones:** Efectos sutiles al hacer scroll y al interactuar con los elementos (3D Tilt Effect).

## Tecnologías Utilizadas

*   HTML5 Semántico
*   CSS3 (Variables, Flexbox, CSS Grid)
*   Vanilla JavaScript (DOM Manipulation, Intersection Observer)

## Estructura de Archivos

*   `index.html`: La estructura principal de la landing page.
*   `css/style.css`: Toda la hoja de estilos, paleta de colores (Verde Oliva, Terracota, Sage) y diseño responsivo.
*   `js/main.js`: Lógica del sistema bilingüe, renderizado del catálogo, filtros, modales y animaciones.
*   `assets/`: Carpeta con imágenes, logos (SVG) y recursos gráficos.
